The plugin includes all possibilities of a social network facebook. 

== Description ==

Plugin add in your posts and/or pages like button, facepile, comments form, and in sidebar activity feed, recommendations, like box, livestream and login button by Facebook. You can configure all options. <a href="http://bogutsky.ru">More information</a>.

== Installation ==
1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
